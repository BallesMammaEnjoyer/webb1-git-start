Arbetsflöde/gång

  1. Skapa en fork av ett repository
  2. klonade repot med github desktop(gröna knappen)
  3. öppnade repot i visual studio code
  4. Öppnade filen index.html, skapade listor och bilder
  5. För att visa bilden så gjorde vi en images mapp och sparade bilden i den
  6. Bilden var gigantisk, vi öppnade den i paint och minskade storleken
  7. Vi gjorde en fin commit med ett begrepligt meddelande
  8. Vi pushade allt till Github
  9. Vi laddade även in en google font
  10. Adda fonten med pluset och lägg sedan till länken i head taggen
  11. Sist så lägger vi till font-family i css filen
  12. För att spara denna lista på github behöver jag nu göra en commit och sedan push
  13. Profit $$$$$
